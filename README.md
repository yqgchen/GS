GS: Gradient Synchronization for Multivariate Functional Data
====

This package provides implementation of gradient synchronization for multivariate functional data, proposed by  
Chen, Y., Lin, S.-C., Zhou, Y., Carmichael, O., MÃ¼ller, H.-G. and Wang, J.-L. (2023+). Gradient Synchronization for Multivariate Functional Data, With Application to Brain Connectivity. Journal of the Royal Statistical Society, Series B (Statistical Methodology). (accepted).

## Installation of the current development version
You can install the development version of the package in R using:
```
devtools::install_github("yqgchen/GS")
```

## Load Package and Data
Once installed you can load the package with:
```
library(GS)
```
